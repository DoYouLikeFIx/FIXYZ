# Risk Acceptance Records

Use this directory only when Epic 12 (DMZ hardening) work is intentionally deferred from Medium to Low priority.

## Naming Rule

- File name: `<YYYYMMDD>-epic12.md`

## Minimum Record Fields

- Decision date (UTC)
- Decision owner
- Deferred stories (12.x list)
- Accepted risks and impact
- Mitigations in place
- Revisit date
